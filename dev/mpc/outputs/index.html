MPC Outputs

REopt.mpc_results — Function

MPC Scenarios will return a results Dict with the following keys:

  • ElectricStorage
  • HotThermalStorage
  • ColdThermalStorage
  • ElectricTariff
  • ElectricUtility
  • PV
  • Generator
